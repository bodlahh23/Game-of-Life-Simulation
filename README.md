# Game-of-Life-Simulation
This project implements Conway’s Game of Life using R. It simulates multi-generational cellular automata and visualizes dynamic grid evolution using ggplot2.
# Features
- Neighbor-count algorithm implementation
- Multi-generation simulation
- Grid visualization using ggplot2
- Randomized and customizable initial states
- Modular function-based design

# Computational Concepts Demonstrated
- Cellular automata
- Algorithmic modeling
- State transitions
- Simulation design
- Data visualization

# Technologies Used
- R
- ggplot2
- tidyverse

# How to Run
Open src/game_of_life.R in RStudio and execute:
runSimulation(initial_matrix, generations=20)
# Future Improvements
- Shiny interactive UI
- Performance optimization for large grids
- Parallel computation
- Pattern detection (gliders, oscillators)
